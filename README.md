# videoapp
Video app made with Node.JS

## about
A simple app that allows people to create a public screen on a beamer and control it with "remotes".
Demo: http://tjallingt.ninja
Inspired by http://protu.be/

## build with:
* Node.js
  * Express
  * Socket.io
* jQuery
* jQuery UI
* perfect-scrollbar
